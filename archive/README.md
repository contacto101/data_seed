# Archivo DataSeed

Este directorio conserva material que ya no debe vivir en la raíz activa del repo, pero que puede ser útil para trazabilidad.

## Subdirectorios

- `legacy/`: documentos reemplazados por fuentes maestras activas.
- `testing/`: archivos de prueba o acceso movidos fuera de raíz.

## Regla

Archivar antes de borrar cuando exista duda. Eliminar solo cuando el historial Git y el índice documental sean suficientes.
