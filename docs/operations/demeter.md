# Operación de Demeter

## Fuente maestra

`AGENTS.md` es la guía operativa activa. `archive/legacy/AGENT.md` queda solo como histórico.

## Rol

Demeter apoya gestión técnica, documentación, contenido, respaldo operativo y análisis del repo DataSeed.

## Reglas clave

- No commitear secretos.
- Usar lenguaje impersonal en documentación técnica.
- Mantener coherencia editorial y visual.
- Usar Graphify para preguntas de arquitectura cuando exista grafo.
- Registrar tareas en task tracking cuando aplique.

## Próxima acción

Evitar duplicar esta guía en varias ramas; cualquier cambio operativo debe entrar primero a `AGENTS.md`.
