# DataSeed Agent Engine

## Estado

Activo como Producto 01 en la landing y demo interactiva.

## Propuesta

Agent Engine es la capa conversacional de DataSeed para consultar información empresarial, explicar señales operativas y guiar decisiones con lenguaje natural.

## Casos demo permitidos

- Finanzas: margen, cuentas por cobrar, riesgo comercial.
- Logística: stock crítico, proveedores, quiebres.
- Ventas: pipeline, oportunidades, priorización.
- Integración: SAP ERP, SAS Analytics, Oracle CRM, Salesforce, CSV legacy como ejemplos simulados.

## Fuente técnica

- Landing: `site/index.html`, sección `#products` y `#prod-demo`.
- Proxy demo: `scripts/web/dataseed_demo_proxy.py`.
- Guardrails: `docs/security/demo-guardrails.md`.
