# Roadmap DataSeed

## Estado

Borrador activo. Este roadmap organiza lo detectado por el grafo multi-branch y la landing actual.

## Fase 1 — Ordenar el repositorio

- Definir fuentes maestras.
- Separar docs/product, docs/commercial, docs/operations y docs/security.
- Mover sitio a `site/` y scripts a `scripts/ops` / `scripts/web`.
- Mantener wrappers de compatibilidad.

## Fase 2 — Producto demostrable

- Consolidar Agent Engine como demo segura.
- Definir ficha técnica y comercial.
- Probar guardrails de demo pública.

## Fase 3 — Pública MVP

- Definir fuentes ChileCompra.
- Diseñar scoring inicial.
- Armar primer flujo de oportunidad priorizada.
- Preparar pitch para empresas B2B chilenas.

## Fase 4 — Plataforma privada

- Revalidar auth GCP/Firebase.
- Evitar assets JSON privados públicos.
- Diseñar API segura si hay datos internos.

## Fase 5 — Operación robusta

- Mantener backup diario y task tracking.
- Regenerar Graphify tras cambios grandes.
- Revisar cronjobs después de cada reorganización estructural.
